Documentation

This API is built to help provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The endpoints

  • /movies
  • /movies/[Title]
  • /genres/[Name]
  • movies/directors/[Name]
  • /users
  • /users/[Username]
  • /users/[Username]//movies/[movie_id]

The Methods available for each endpoint

  • /movies: GET : returns a list of movies available on myFlix
  • /movies/[title]: GET : to see the description of a movie by title. Eg: http://localhost:8080/movies/Aladdin
  • /genres/[Name]: GET : to see the description of a genre by name. Eg:http://localhost:8080/genres/Comedy
  • movies/directors/[Name]: GET : to get the description of a director by name. Eg: http://localhost:8080/directors/Tim%20Burton
  • /users: POST : add a new user with JSON format in request body. Eg: { "Username": "DancingQueen21", "Password": "Iloveme", "Email": "DianaK24@live.com", "Birthday": "1988-04-13T00:00:00.000Z" }
    New user data including user ID in JSON format is the response to the client.
  • /users/[Username]: PUT : updates user info by Username. Eg: First go to URL: http://localhost:8080/users/DancingQueen21, then type JSON format in request body :{ "Username": "DancingQueen21", "Password": "Ilovemeandyou", "Email": "DianaK24@live.com", "Birthday": "1988-04-13T00:00:00.000Z" }
    Updated user data in JSON format is the response to the client.
  • /users/[Username]: DELETE : deletes a user (deregister) by user id. Eg: http://localhost:8080/users/DancingQueen21. The response is a message indicating user has been deleted
  • /users/[Username]/movies/[movie_id]: POST : Adds a movie to favorites. Eg: http://localhost:8080/users/layla234/movies/5d30775be25340d93bbb6b45. A user data in JSON format including newly added movie (ID) is the response to the client.
  • /users/[Username]/movies/[movie_id]: DELETE : Removes a movie from favorites. Eg: http://localhost:8080/users/Pravinth/movies/5d30775be25340d93bbb6b45. Updated user data in JSON format with requested movie removed from the user's favorite movies list is the response to the client.
