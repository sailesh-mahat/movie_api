Documentation

This API is built to help provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The endpoints

  • /
  • /movies
  • /movies/[title]
  • /Genre
  • /Genre/[name]
  • /Director
  • /Director/[name]
  • /users
  • /users/[id]
  • /users/[id]/[movie_id]

The Methods available for each endpoint

  • /: GET : takes you to the homepage of myFlix
  • /movies: GET : returns a list of movies available on myFlix
  • /movies/[title]: GET : to see the description of a movie by title. Eg: http://localhost:8080/movies/Aladdin
  • /Genre: GET : to see the available genres
  • /Genre/[name]: GET : to see the description of a genre by name. Eg:http://localhost:8080/genres/Comedy
  • /Director: GET : to see the list of all directors
  • /Director/[name]: GET : to get the description of a director by name. Eg: http://localhost:8080/directors/Tim%20Burton
  • /users: GET : returns a list of all users.
  • /users: POST : add a new user with JSON format in request body. Eg: { "name": "Kritika", "username": "kitkat", "password": "", "email": "abc@xyz.com", "birthday": "", "favorites": [] }
  • /users/[id]: PUT : updates user info by id. Eg: First : http://localhost:8080/users/042e5237-de6a-48f1-aea2-922fc3174259, then JSON format in request body : { "name": "Sailesh", "email": "azs@ffn.com", "favorites": [4] }
  • /users/[id]: DELETE : deletes a user (deregister) by user id. Eg: http://localhost:8080/users/042e5237-de6a-48f1-aea2-922fc3174259
  • /users/[id]/[movie_id]: POST : Adds a movie to favorites. Eg: http://localhost:8080/users/042e5237-de6a-48f1-aea2-922fc3174259/6
  • /users/[id]/[movie_id]: DELETE : removes a movie from favorites. Eg: http://localhost:8080/users/042e5237-de6a-48f1-aea2-922fc3174259/6
